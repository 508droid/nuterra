# Install instructions

1. Replace `/TerraTechWin64_Data/Managed/Assembly-CSharp.dll` with the (same-named) dll provided in the release.
2. Start the game

# Configuration

The mod automatically creates a configuration file in the following location: `/maritaria/mod.settings`.
In this file you can edit the settings to your own likings. A restart of the game is required to apply changes to the settings.
